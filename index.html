tLock

A locking plugin (chests, doors, signs, etc.) for tShock

Download this project as a .zip file Download this project as a tar.gz file

Welcome to tLock.

tLock is the most user-friendly locking program for tShock. With a nice, friendly, in-game gui, complete with buttons and a text box, you'll never confuse users with a "type /lockchest password 105cd" or any command, for that matter. With easy, built-in permissions and the option to change them (for more advanced users), tLock will secure your Terraria server, easily, the second it's installed.
